Spatiotemporal LGCPs

Chapter 2 Hawkes process

A univariate Hawkes process is defined to be a self-exciting temporal point process where the conditional intensity function is given by

\[\lambda(t)= \mu(t)+ \Sigma_{i:\tau_i<t}\nu(t-\tau_i)\] where where \(\mu(t)\) is the background rate of the process and \(\Sigma_{i:\tau_i<t}\nu(t-\tau_i)\) is some historic temporal dependence.

Data are recorded line lists of individuals with malaria (caused by the Plasmodium vivax parasite) in Yunnan Province, China between 1st January 2011 to 24th September 2013. 2.3 Simulation

Set parameters

mu <- 0.25
alpha <- 0.13
beta <- 0.15

Option 1

Simulation algorithm as per On Lewis’ simulation method for point processes

sim1 <- sim_hawkes(mu = mu, alpha = alpha, beta = beta, plot = TRUE)

Option 2

Simulation algorithm as per MLE of Hawkes’ self-exciting point process

sim2 <- sim_hawkes(mu = mu, alpha = alpha, beta = beta,n = 250, method = "2", plot = TRUE)

Option 3

Using the TMB likelihood template; below we simulate another realisation from the fitted china_malaria model:

sim3 <- fit$simulate()
