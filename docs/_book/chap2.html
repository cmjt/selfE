2.2.3 Modelling malaria data

Here we consider data from Using Hawkes Processes to model imported and local malaria cases in near-elimination settings. Data are recorded line lists of individuals with malaria (caused by the Plasmodium vivax parasite) in Yunnan Province, China between 1st January 2011 to 24th September 2013. Data may be downloaded from here as follows.

tmp <- tempfile(fileext = ".RData")
download.file("https://dataverse.harvard.edu/api/access/datafile/4443458",
              destfile = tmp)
load(tmp)
unlink(tmp)
str(china_malaria)

2.3 Simulating a Hawkes process

Set parameters

mu <- 0.25
alpha <- 0.13
beta <- 0.15

2.3.1 Option 1

Simulation algorithm as per On Lewis’ simulation method for point processes

sim1 <- sim_hawkes(mu = mu, alpha = alpha, beta = beta, plot = TRUE)
## `stat_bin()` using `bins = 30`. Pick better value with `binwidth`.

2.3.2 Option 2

Simulation algorithm as per MLE of Hawkes’ self-exciting point process

sim2 <- sim_hawkes(mu = mu, alpha = alpha, beta = beta,n = 250, method = "2", plot = TRUE)
## `stat_bin()` using `bins = 30`. Pick better value with `binwidth`.
